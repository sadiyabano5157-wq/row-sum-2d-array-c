# row-sum-2d-array-c
A simple C program that calculates and displays the sum of each row in a 2D array.

# Row Sum of 2D Array (C Program)

## 📌 Description
This program calculates the sum of each row in a 3×3 two-dimensional array.

## 🚀 How It Works
- Store values in a 2D array.
- Traverse each row using nested loops.
- Calculate the sum of elements in each row.
- Display the row-wise sums.

## 📥 Input Array

```text
1 2 3
4 5 6
7 8 9
```

## 📤 Output

```text
Sum of row 1 = 6
Sum of row 2 = 15
Sum of row 3 = 24
```

## 🛠️ Concepts Used
- 2D Arrays
- Nested Loops
- Row-wise Traversal
- Sum Calculation

## 🎯 Learning Outcome
- Understanding 2D arrays
- Working with nested loops
- Row-wise processing of matrices
